# LuxeSmile Dental Clinic

Welcome to the **LuxeSmile Dental Clinic** website repository. This is a premium, modern, and fully responsive landing page designed for a high-end dental practice. It features a clean, professional design, smooth micro-animations, and a user-friendly interface optimized for converting visitors into patients.

## 🌟 Key Features

- **Modern & Premium Design:** A clean, vibrant, and highly aesthetic interface that instills trust and professionalism.
- **Fully Responsive Layout:** Optimized for all devices, from desktop screens to mobile phones.
- **Interactive Elements:** Smooth scroll navigation, hover effects, and animated sections using CSS and vanilla JavaScript.
- **Hero Section:** Features a compelling call-to-action (CTA), dynamic statistics, and floating action cards (AI Assistant, Smile Preview).
- **Service Showcases:** Beautifully presented cards detailing all available dental services.
- **Patient Transformations:** An engaging results carousel showing real before-and-after cases.
- **Why Choose Us:** Highlights clinic benefits like pain-free tech, AI smile simulation, and a luxury lounge.
- **Doctors Profile:** Introduces the specialist team with their professional credentials and social links.
- **Booking Integration:** An intuitive appointment booking form for seamless patient onboarding.

## 🛠️ Technology Stack

- **HTML5:** Semantic and accessible structure.
- **CSS3 (Vanilla):** Custom styling, variables for easy theme changes, flexbox/grid layouts, and animations.
- **JavaScript (Vanilla):** Handling mobile menu, smooth scrolling, scroll-based animations, and dynamic counters.

## 🚀 Getting Started

### Prerequisites

You only need a web browser to view this site locally. No complex build tools are required.

### Installation & Setup

1. **Clone the repository:**
   ```bash
   git clone https://github.com/ahmed-ali-codes/luxesmile-dental-site.git
   ```
2. **Navigate to the directory:**
   ```bash
   cd luxesmile-dental-site
   ```
3. **Open `index.html`:**
   Simply double-click the `index.html` file to open it in your default web browser, or use a tool like Live Server in VSCode for a better development experience.
